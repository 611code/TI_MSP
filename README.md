# TI板 模块移植全攻略

更新中........

基于[【立创·天猛星MSPM0G3507开发板】介绍 | 立创开发板技术文档中心](https://wiki.lckfb.com/zh-hans/tmx-mspm0g3507/)

本项目主要重点在于模块的移植，想详细了解原理可以看立创开发指南。

其他侧重点有(在API汇总里)：

GPIO控制
I2C读写
SPI读写
us延时/ms延时
UART收发
TIM/PWM设置
TIM/编码器读取

## PS

使用 J-link 烧录，有些 J-link 烧录需要对单片机单独供电，如果遇到无法连接的情况可以试一试。
